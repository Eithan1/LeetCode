# 题目描述：合并区间
给出一个区间的集合，请合并所有重叠的区间。

示例 1:

    输入: [[1,3],[2,6],[8,10],[15,18]]

    输出: [[1,6],[8,10],[15,18]]

    解释: 区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].

示例 2:


    输入: [[1,4],[4,5]]

    输出: [[1,5]]

    解释: 区间 [1,4] 和 [4,5] 可被视为重叠区间。
    
# 算法
首先将所有区间按start递增排序

先将第一个存入结果

然后依次判断下一个与结果中的最后一个区间能否合并

可以则合并

   - start不变（因为start已按递增排序）
   - end变为两个区间的较大的end

不行则将区间添加到结果中
