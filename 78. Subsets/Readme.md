# 题目描述：子集

给定一组不含重复元素的整数数组 nums，返回该数组所有可能的子集（幂集）。

说明：解集不能包含重复的子集。

示例:


    输入: nums = [1,2,3]

    输出:

    [
  
          [3],
  
          [1],
  
          [2],
  
          [1,2,3],
  
          [1,3],
  
          [2,3],
  
          [1,2],
  
          []

    ]
    
# 算法
首先结果里存放空集[]

当读入nums中的一个数1，将1放入[]，得到[1]，再加上之前的空集得到[],[1]

再从nums中读入一个数2，将2分别放入已经得到的[],[1]，得到[2],[1,2]，再加上之前的得到[],[1],[2],[1,2]...
